Follow The Flash!

BRIEF: Loosely based on the 'Simon Says' game.

Follow The Flash! is a one player memory game loosely based on Simon Says. This was my first project to build on the Web Development Immersive course at General Assembly using JavaScript, jQuery, HTML and CSS.

Tech used....... HTML, CSS, jQuery & JavaScript

SnowFo

BRIEF: A list of countires I have skied.

The task was to build a multi-page app using authentication with a bonus of adding an API. SnowFo list's the best ski resorts that I have skied in Europe and Canada. I have also added an 'Add' feature where users can add their own resorts where they have skied. I used Balsamic as the wire frame and trello to help me plan.

Tech used....... HTML, CSS, SCSS, jQuery, JavaScript, Express.js, Node.js, MongoDB & Mongoose

You can view the app HERE, and the code on Github.

Runch

BRIEF: As a team, build a MEAN Stack application with an API and authentication.

RUNCH is an app targeted at getting people, who have little spare time, active. It enables them to use their lunch-time wisely - they can go for a quick run and meet other like-minded professionals. It is an opportunity to not only get fit, but also to network, and feel refreshed for the second half of the day..

Tech used....... HTML, CSS, SCSS, JavaScript, AngularJS, Express.js, Node.js, MongoDB & Mongoose

You can view the app HERE, and the code on Github.
